# Contributing to Canasta Familiar

¡Gracias por tu interés en contribuir a Canasta Familiar!

## Código de Conducta

Este proyecto y todos los participantes están bajo el Código de Conducta de [Contributor Covenant](https://www.contributor-covenant.org). Al participar, se espera que mantengas este código.

## ¿Cómo Contribuir?

### Reportar Bugs

Si encontraste un bug, por favor abre un issue en GitHub con:
- Una descripción clara del bug
- Pasos para reproducirlo
- Tu entorno (SO, versión de Node, etc.)
- Screenshots si es relevante

### Sugerir Funcionalidades

¿Tienes una idea para una nueva característica? ¡Abre un issue! Incluye:
- Descripción clara de la funcionalidad
- Por qué sería útil
- Si es posible, algunos ejemplos de uso

### Pull Requests

1. Fork el repositorio
2. Crea una rama para tu feature (`git checkout -b feature/nueva-caracteristica`)
3. Haz tus cambios
4. Asegúrate de que los tests pasen
5. Commit con mensajes claros
6. Push a tu fork
7. Abre un Pull Request

### Estándares de Código

- Usa TypeScript
- Sigue las convenciones de nombres existentes
- Comenta código complejo
- Actualiza la documentación si es necesario

### Proceso de Review

- Los maintainers revisarán tu PR
- Puede que pidamos cambios
- Una vez aprobado, mergearemos tu PR

- GitHub Issues: Bugs y features
